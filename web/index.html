SETUP GUIDE

loopMIDI

  1. Download & Install loopMIDI from here.
  2. Right-click loopMIDI and "Run as Administrator".
  3. Create a port named: 'FL-Gesture-Port-BETA'
    (leave this app running in the background).

FL Studio

Once the app opens on your web browser:

  1. (BEFORE OPENING FL Studio)
    Drag the 'Gesture Controller' folder into your
    FL Studio's Hardware folder (found in
    'Image-Line\FL Studio\Settings\Hardware').
  2. Run FL Studio and go to 'Options\MIDI Settings'.
  3. Select 'FL-Gesture-Port-BETA' in the Output Window.
  4. In the Input Window, select the same port and set the Controller type to 'Gesture Controller' (if folder is dragged in correctly).

You can now use the Gesture Controller!
