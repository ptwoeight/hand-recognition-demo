# Setup
1. Run `install.bat` for app dependencies.

2. Copy `GestureController` folder into `Documents\Image-Line\FL Studio\Settings\Hardware`.
    > This is a translation script for gestures to be able to send MIDI signals!

3. Open FL Studio and navigate to `Options > MIDI Settings`.
4. Select your MIDI Input Device (e.g. 'GesturePort 1').
5. Set the **Controller Type** to 'Gesture Controller Script'.
6. Click **Enable**.
